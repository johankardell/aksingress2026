# Security Policy

## Supported Use

This repository is an educational AKS ingress demo. It is not intended to be a production-ready reference architecture without additional hardening and review.

## Reporting a Vulnerability

Please report security issues by opening a private vulnerability report on GitHub, or contact the repository maintainer directly if private reporting is unavailable.

Do not include secrets, tenant-specific identifiers, or other sensitive data in public issues or pull requests.
