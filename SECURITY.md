# Security Policy

## Supported Scope

This repository is an educational AKS ingress comparison demo. It is not a production product or service, and it does not provide formal support commitments.

Security guidance applies to the demo infrastructure templates, deployment scripts, Kubernetes manifests, and sample application included in this repository.

## Reporting a Vulnerability

If you find a security issue in this repository:

1. Open a GitHub issue if the details are safe to discuss publicly.
2. If public disclosure would create risk, contact the repository maintainer privately before sharing exploit details.
3. Include the affected file or demo, impact, reproduction steps, and any suggested mitigation.

Please do not include secrets, subscription identifiers, or other sensitive environment details in public issues.

## Expectations

Because this is a demo repository, fixes are handled on a best-effort basis. For vulnerabilities in Azure, AKS, Kubernetes, ingress controllers, or other upstream products, report issues through the relevant vendor or project security process.
